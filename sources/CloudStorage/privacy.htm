Privacy policy

Program is not gathering or storing private user information.

Cloud storage is accessing Google Drive authenticated by user. Permissions are limited only to program's application data space and used to partially mirror FFTriadBuddy-settings.json file. User can revoke access and/or remove application data at any time using Google Drive's settings.

If you have any concerns, just check source code at https://github.com/MgAl2O4/FFTriadBuddy
